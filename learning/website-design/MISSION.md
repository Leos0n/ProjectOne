# Mission: Website & UI Design

## Why
Go from beautiful AI-generated design images to real, working products. Generate stunning UI concepts with image models (Grok Imagine, ChatGPT image), then turn them into live websites — and later iPad/phone/laptop apps — for business or play, directing Claude Code as the implementer.

## Success looks like
- Can look at any website and articulate *why* it looks good or bad (hierarchy, spacing, typography, color)
- Can write image-generation prompts that produce premium, implementable UI mockups (one section per image)
- Has shipped at least one real, deployed website that started as an AI-generated image and looks faithful to it
- Can direct Claude Code with precise design vocabulary instead of "make it look better"
- Repeatable pipeline: concept → mockup images → critique → implementation → deploy

## Constraints
- Beginner / non-coder: Claude Code writes the code; the learner directs design and learns to read/tweak HTML & CSS, not write it from scratch
- Web first; native iPad/phone apps are a later phase
- Tools: Claude Code with the taste-skill pack (installed in this branch at `.claude/skills/`), Grok/ChatGPT for image generation
- Time budget shared with other learning topics — lessons must be short and project-driven

## Out of scope (for now)
- Native app development (SwiftUI / React Native) — phase 2, after the web pipeline works
- Backend development, databases, auth
- Figma/design-tool mastery — images come from AI generation, not manual design tools

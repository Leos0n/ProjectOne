# Spacing lever internalized; hierarchy vs typography still blurry

Completed lesson 0001. In retrieval practice, correctly identified whitespace as the first fix for a cramped, cheap-looking mockup — the spacing lever is landing. However, when shown "everything big, bold, and bright so nothing leads the eye," answered *typography* instead of *hierarchy*. The distinction to reinforce: typography is about how letters are set (typefaces, scale, line length); hierarchy is about which element leads and which recede — emphasis and de-emphasis across the whole page.

**Implications:** lesson 0002 (typography & spacing systems) should open by contrasting the two levers side by side before going deeper into type scales. The three-site critique homework is still pending and will show whether the confusion persists in the wild.
